# Cyber Attack Simulator
An educational Docker-based cybersecurity simulator for testing and understanding common web attacks and defense mechanisms.

## Features

- **DDoS Attack Simulation**: Test distributed denial of service attacks with rate limiting
- **SQL Injection Testing**: Attempt various SQL injection techniques against a vulnerable database
- **XSS Attack Testing**: Cross-site scripting vulnerability detection
- **Port Scanning**: Network reconnaissance simulation
- **Real-time Firewall**: Adaptive defense with rate limiting, IP blacklisting, and pattern detection
- **Live Monitoring**: Real-time attack logs and firewall alerts
